# PHP x Mailjet

## Guide to Run this Project

<br>

### Initialize Project

    composer init

### Install Library - PHPMailer & PHPdotenv

    composer require phpmailer/phpmailer vlucas/phpdotenv

<br>

### On file _.env_ and _sender.php_ - *Change with Your Credential*

<br>

### Run

    php -S localhost:3000
